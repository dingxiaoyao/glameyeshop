# 欧美模特佩戴图 Prompt 包(10 SKU 量身定做)

**任务**:为 10 个 GlamEye DIY Lash Cluster Kit 各生成 1 张"欧美模特佩戴效果图",
作为 product card 的 hover 切换图(用户鼠标移到商品上,主图淡出,模特佩戴图淡入)。

---

## 关键统一规则(每条 prompt 都要满足)

| 维度 | 设定 |
|---|---|
| 视角 | Close-up portrait,单眼或双眼特写 |
| 模特 | 欧美面孔,多元化(西欧 / 北欧 / 拉丁 / 黑人 / 东欧 各 1-2),20-30 岁 |
| 妆容 | 极简 — 干净底妆 + 中性唇,**焦点全在眼睛和睫毛** |
| 睫毛风格 | **cluster lashes 贴在自然睫毛下方**(关键!不是 strip lash 不是 extensions) |
| 背景 | 纯白或米色 #f5f0e8,**柔和散射光**,无阴影杂物 |
| 比例 | `--ar 4:3`(跟 product-image aspect-ratio 4/3 完全匹配) |
| 风格关键词 | luxury beauty editorial, Vogue cover style, hyperreal skin, soft natural light |
| 工具 | **Midjourney v6 主推**,Flux 1.1 Pro 次之 |

---

## 全局 Negative Prompt(每条都加)

```
strip lashes, lash extensions on top of lid, glue band visible, fake plastic
look, cartoon, illustration, mascara clumps, heavy eyeliner, low quality,
blurry, overprocessed, instagram filter, ar 1:1, square
```

## 推荐流程(在 Midjourney 里)

### 步骤 1:跑第一张(GE-CK-NATURAL)

直接复制上面的 prompt 到 Midjourney,加 negative。Midjourney 会出 4 张,挑最满意的 U(upscale)。

### 步骤 2:获取 character reference

把那张 upscale 后的图保存,复制图链接。后续 9 个 SKU 不一定要用同一模特(我们要的就是多元化),但**整体灯光/质感/角度**要一致。可以用这张作 **style reference**:

```
[你想生成的 SKU prompt] --sref [前一张图的 URL] --sw 100
```

`--sref` 是 style reference,`--sw 100` 是风格权重(0-1000,100 是 medium)。

### 步骤 3:批量跑

10 张大约 30-60 分钟(包含 reroll 挑选)。Midjourney 标准包($10/月)每月 200 次生成额度,够跑 50+ 张挑 10 张。

## 不想跑 Midjourney 的备选方案

如果你没用过 AI 图像工具或者不想注册:

- **fal.ai** — 网页端,Flux 1.1 Pro 每张 ~$0.04,无需订阅,即开即用
- **Krea.ai** — 实时调整,$10/月,Beauty 编辑界面很友好
- **Pikaso(由 Freepik 出)** — 免费额度,质量也不错
- **ComfyUI(本地)** — 免费,但需要 NVIDIA GPU

我推荐 fal.ai 起步 — 完全无门槛,出 10 张大概 $0.5。注册 → 选 Flux Pro → 把上面的 prompt 粘进去 → 跑。
